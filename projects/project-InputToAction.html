Input To Action

A 2D puzzle platformer created for GMTK Game Jam 2023 (theme: "Roles Reversed"), placing in the TOP 5%.

YouTube itch.io

Overview

Input To Action is a 2D puzzle platformer built in 48 hours for GMTK Game Jam 2023, under the theme "Roles Reversed". The game placed in the TOP 5% out of thousands of entries. The core concept inverts the relationship between player input and character action, challenging players to rethink how they approach movement and interaction.

Core Mechanics

  • Inverted input system — actions are tied to unexpected controls
  • Puzzle-platformer level design built around the role-reversal theme
  • Progressive difficulty scaling across handcrafted levels

Technical Notes

Built with Unity and C# within a 48-hour jam constraint. Focus was on rapid prototyping, tight controls, and ensuring the core mechanic communicated clearly to players without tutorials.
