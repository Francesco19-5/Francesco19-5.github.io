Project Cyberpunk

A solo Unreal Engine 5 project exploring cyberpunk aesthetics, systems design and environment building.

YouTube

Overview

Project Cyberpunk is a solo project built in Unreal Engine 5, focused on creating a dense cyberpunk environment with polished visuals and interactive gameplay systems. The project explores lighting, materials, and Blueprint-driven gameplay.

Core Systems

  • Cyberpunk environment design with Lumen global illumination
  • Blueprint-driven gameplay and interaction systems
  • Custom materials and post-processing for the neon aesthetic

Technical Notes

Built entirely in Unreal Engine 5 using Blueprints. Focus was on environmental storytelling, visual fidelity, and building reusable modular systems for future projects.
