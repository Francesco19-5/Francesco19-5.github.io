Project Cyberpunk

A group project creating a Cyberpunk 2077-style mission — full gameplay implementation with procedural weapon systems and enemy AI.

YouTube

Overview

Project Cyberpunk is a group project built in Unreal Engine 5, creating a mission in the style of Cyberpunk 2077. The team implemented the core gameplay loop from scratch — including combat, weapon handling, and enemy AI — set within a dense, neon-lit urban environment.

My Role

I was responsible for the weapon gameplay systems: designing and implementing the full gun pipeline, and building a procedural recoil system that calculates kick direction and intensity per weapon. I also handled the design and documentation of the enemy AI.

Core Systems

  • Procedural recoil system — direction and intensity generated per-shot based on weapon parameters
  • Weapon gameplay pipeline: firing, hit detection, visual and audio feedback
  • Enemy AI design: behavior states, perception, and combat reactions

Technical Notes

Built in UE5 using Blueprints. The procedural recoil system is fully data-driven — each weapon exposes parameters for kick strength, recovery speed and spread pattern, allowing designers to tune feel without touching code.
