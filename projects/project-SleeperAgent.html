Sleeper Agent

A stealth-focused game design project with a detailed Game Design Document and full mechanics documentation.

YouTube

Overview

Sleeper Agent is a stealth action game concept developed with a complete Game Design Document. The project focuses on designing systems for player infiltration, AI awareness, and mission structure — inspired by games like Splinter Cell.

Core Systems

  • Stealth mechanics — visibility, noise, and cover systems
  • AI awareness states and patrol behavior design
  • Mission structure with optional objectives
  • Full GDD documenting all mechanics and design decisions

Design Notes

This project was an exercise in thorough pre-production documentation — designing a complete, playable stealth game on paper before moving to implementation, including edge cases and player progression.
