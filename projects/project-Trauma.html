Trauma

A narrative-driven game exploring trauma through gameplay systems and state machine-driven design.

YouTube itch.io

Overview

Trauma is a solo Unreal Engine 5 project that explores how game mechanics can express psychological states. Built with a complex state machine at its core, the game uses player-controlled systems to mirror the experience of navigating trauma.

Core Systems

  • State machine-driven gameplay — character states influence mechanics
  • Narrative systems that react to player choices and progression
  • Custom Blueprint + C++ architecture for the state machine
  • Audio and visual feedback tied to state transitions

Technical Notes

The state machine was implemented in a hybrid Blueprint/C++ approach to balance rapid iteration with performance. Careful attention was given to ensuring state transitions felt meaningful and communicated clearly to the player.
