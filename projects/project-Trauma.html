Trauma

A survival-horror game exploring the trauma of the main character — escape the nightmare and unravel the mystery.

YouTube itch.io HLGDD

Overview

Trauma is a group survival-horror project built in Unreal Engine 5. The player steps into the nightmare of the main character, navigating an environment shaped by psychological trauma while evading enemies and uncovering the mystery behind it all.

My Role

As Game Designer I was involved across the full development lifecycle: ideation, prototyping, and testing. My specific contributions included:

  • AI documentation for the main enemy and minions — from concept to final behavior spec
  • Collaborative work on the core game mechanics with the team
  • Full Animation Blueprint setup in UE5, covering all character animations
  • Writing and maintaining the project documentation throughout development

Core Systems

  • Enemy AI — main antagonist with complex awareness and pursuit behavior
  • Minion AI — supporting enemies with their own behavior states
  • Animation Blueprint system driving all character animations in UE5
  • Narrative-driven level design guiding the player through the trauma arc

Technical Notes

Built in UE5 with a hybrid Blueprints + C++ architecture. The Animation Blueprint covers all characters and was set up to be both robust and easy to extend. AI behavior was fully documented before implementation, keeping design intent and technical execution aligned throughout the project.
