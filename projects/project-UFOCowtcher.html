UFO Cowtcher

A mobile group project — pilot a UFO over a farm and abduct as many cows as you can before time runs out.

YouTube itch.io HLDD

Overview

UFO Cowtcher is a mobile game built as a group project in Unity. The player controls a UFO using touch input, flying over a farm and activating a tractor beam to abduct cows before time runs out. The game, inspired by UFO Catcher, leans into a lighthearted, absurd tone with satisfying feedback at every catch.

My Role

As Game Designer I contributed across all phases — ideation, prototyping and testing. My specific work included:

  • Running and writing post-playtest feedback reports, collecting and synthesising tester input
  • UI implementation in-game
  • Writing the description and personality for each cow in the game

Core Mechanics

  • UFO movement with mobile-friendly touch controls
  • Tractor beam abduction with timing-based interactions
  • Score system with escalating difficulty
  • Audio and visual feedback per cow catch, reinforcing the game's tone

Technical Notes

Developed in Unity targeting mobile platforms. Focus on smooth touch input, lightweight assets, and a tight game loop that stays engaging across short play sessions. The High Level Design Document is available to download above.
