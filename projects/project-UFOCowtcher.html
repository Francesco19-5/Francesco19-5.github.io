UFO Cowtcher

A mobile game jam entry where you pilot a UFO and abduct as many cows as possible — fast-paced and fun.

YouTube itch.io

Overview

UFO Cowtcher is a mobile game jam entry built with Unity. The player controls a UFO using touch/tilt input, flying over a farm and activating a tractor beam to abduct cows before time runs out. The game prioritizes satisfying feedback loops and a lighthearted tone.

Core Mechanics

  • UFO movement with mobile-friendly touch controls
  • Tractor beam mechanic with timing-based abduction
  • Score system with escalating difficulty
  • Responsive audio and visual feedback for each catch

Technical Notes

Developed in Unity targeting mobile platforms. Emphasis on smooth touch input, lightweight assets, and a tight game loop that stays engaging across short play sessions.
